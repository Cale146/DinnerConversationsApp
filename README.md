Dinner Conversations is an Android application designed to spark meaningful and engaging discussions. It provides a curated collection of thought-provoking questions to help you connect with your kids, friends, and loved ones during dinner or any shared moment. Whether you’re looking to deepen relationships, start interesting conversations, or simply have fun together, this app makes it easy to get people talking.

Features

• Many creative, open-ended questions

• Great for families, friends, dates, and group settings

• Simple, clean, and intuitive interface

• Works completely offline - no ads, no tracking

How to Run the App

Clone this repository:
git clone git@github.com:Cale146/DinnerConversationsApp.git

Open the project in Android Studio.

Let Gradle sync and build automatically.

Run the app on your Android device or emulator.

Requirements

• Minimum Android SDK 21+
